🚀 Team Task Manager

A full-stack team collaboration and task management platform built with the MERN Stack. Designed to help teams organize work efficiently through task assignments, progress tracking, role-based access control, comments, and file sharing.

🌟 Features

👥 User & Team Management

* Secure user authentication and authorization
* Role-based access (Admin & Team Members)
* Team collaboration workspace
* User profile management

📋 Task Management

* Create, update, and delete tasks
* Assign tasks to team members
* Set task priorities and deadlines
* Track task status (To Do, In Progress, Completed)
* Task progress monitoring

💬 Collaboration

* Task-specific comments and discussions
* Real-time team communication workflow
* Activity tracking and updates

📎 File Attachments

* Upload and manage task-related files
* Centralized document sharing
* Easy access to project resources

📊 Dashboard & Analytics

* Task overview dashboard
* Team productivity insights
* Deadline monitoring
* Project status visualization

🛠️ Tech Stack

Frontend

* React.js
* React Router
* Tailwind CSS
* Axios

Backend

* Node.js
* Express.js

Database

* MongoDB
* Mongoose

Authentication

* JWT Authentication
* Role-Based Access Control (RBAC)

Deployment

* Frontend: Vercel
* Backend: Render
